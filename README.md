# Group Theory & Spectroscopy — M.Sc. Chemistry Paper IX

Interactive educational website for complete exam preparation of **M.Sc. Chemistry Paper IX: Group Theory and Spectroscopy**.

Built with Next.js 16, TypeScript, Tailwind CSS 4, Framer Motion, and shadcn/ui.

## Features

- **3 Complete Units** with detailed Hinglish explanations
- **8 Interactive Diagrams** — SVG/CSS animated with Framer Motion
- **50+ PYQs** (2016–2024) with frequency tracking and importance badges
- **Quick Revision** section with key formulas and definitions
- **Dark theme** with emerald/teal accents and gold PYQ markers
- **Fully responsive** — mobile, tablet, desktop
- **Smooth animations** — staggered reveals, scroll-triggered effects, interactive diagrams

## Topics Covered

### Unit I — Molecular Spectroscopy & Group Theory
- Symmetry Elements & Operations
- Point Groups & Flowchart
- Character Tables
- Selection Rules
- IR & Raman Spectroscopy
- Vibrational Spectra (Diatomic & Polyatomic)

### Unit II — Electronic Spectroscopy
- Franck-Condon Principle
- Jablonski Diagram
- Auger Effect & KLL Spectra
- Photoelectron Spectroscopy (PES)
- Charge Transfer Spectra

### Unit III — ESR Spectroscopy
- ESR Principle & Theory
- Hyperfine Coupling
- g-Factor & Anisotropy
- Zero-Field Splitting

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **UI**: shadcn/ui
